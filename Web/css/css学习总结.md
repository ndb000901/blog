# css学习总结

## 选择器(selector)

**元素选择器(element selector)**

```
p {
  color:yellow;
}
```

**类选择器(class selector)**

```
.class {
  color:yellow;
}
```

**id选择器(id selector)**

```
#id {
  color:yellow;
}
```

**通用选择器(universal selector)**

```
* {
  color:yellow;
}
```

**分组选择器(grouping selector)**

```
h1 {
  color:yellow;
}
h2 {
  color:red;
}
```
